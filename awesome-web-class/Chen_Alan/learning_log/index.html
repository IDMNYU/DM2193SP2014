Difference between a pseudo class and a pseudo element

Pseudo classes select regular elements under certain conditions such as position relative to other elements or the state of the element. Single colon. Examples: ":visited", ":active", ":nth-child(n)"

Pseudo elements create new elements beyond the document language. Double colon. Examples: "::after", "::before", "::first-letter"

CSS vendor prefixes allow browsers to add support for new CSS features. Examples: webkit, moz

Box sizing

  • When setting the width you need to account for padding and border size.
  • border-box: width and height is measured with content, padding and border
  • content-box: width and height is measured with content only
  • More Info

Display: none vs Visibility: hidden

  • Display: none will render the page as if the element didn't exist. Commonly used with javascript.
  • Visibility: hidden will hide the element but still take up the space

Learn Layout

Use max-width to handle small windows so there is no scroll bar is the width is bigger then the browser.

Use float: right to push element left or right and have other elements wrap around it.

Use clear: right to not allow floating elements on left/right/both.

Use overflow: auto (clearfix) to specify what to do with the content if it overflows outside the box. Use auto to solve elements that are floated and overflow its containing element.

Use column-count and column-gap to make multicolumn text.

Use flex is another way to do layouts.

Use position: sticky to allow the element go where it would naturally go.

position: absolute will position an element base relative to the parent that is not static.

position: relative will position an element relative to where it would normally lie.
