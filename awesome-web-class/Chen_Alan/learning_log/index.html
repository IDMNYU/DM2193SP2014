Intro to HTML

Code School HTML
  • !DOCTYPE html tells the web browser you are using html
  • There are 6 header types: h1 - h6
  • body tag defines the content of the page
  • html tags with no content are called empty tags such as br
  • Link tags are represented with 'a' and the attribute href defines the link
  • Image tags is an empty tag and the src attribute defines the path or link to the image

html stuff

  • Rule of thirds and grid
  • bold: stylish
  • strong: semantic meaning
  • inline elements: cannot have width or height set. respect left and right margins but not top and bottom
  • block elements: respect top/bottom/right/left break line after block element
  • inline-line elementsL: respect top/bottom/right/left, allows items to sit to the left and right
