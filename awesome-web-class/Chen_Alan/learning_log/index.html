Intro to HTML

The aspect of Structure

  • <!DOCTYPE html> Element tells the web browser that the document is in html
  • There are 6 header types: h1 - h6
  • body tag defines the content of the page
  • html tags with no content are called empty tags such as br
  • Link tags are represented with 'a' and the attribute href defines the link
  • Image tags is an empty tag and the src attribute defines the path or link to the image

Design is important

The aspect of Design

  • Narrative, design and interface matters the user
  • Design
    • Look and Feel

    • Info Architecture
  • Mapping, wayfinding, teaching
  • Div has no semantic meaning
  • ARIA: access right internet applications is to make the web apps for accessible to people with disabilities. Example would be adding the role attribute to tags.
  • Two reasons to use the attribute role is to override the role when there is no host language element that is appropriate
  • The other reason is to back up a native element to support browsers that implement the ARIA role

Semantic Meaning

Blocks, blocks, and more blocks ...

  • Rule of thirds and grid
  • bold: stylish
  • strong: semantic meaning
  • inline elements: cannot have width or height set. respect left and right margins but not top and bottom
  • block elements: respect top/bottom/right/left break line after block element
  • inline-line elementsL: respect top/bottom/right/left, allows items to sit to the left and right
