CSS

Presentation pleases the eyes

There are three ways of adding CSS

  • Adding the style attribute to the tag
  • Inline CSS: adding the style element to the html page
  • External CSS: adding a link element to the html page that references an external CSS stylesheet

Reset CSS reduces inconsistency between browsers by removing all built-in browser styling

Normalize CSS aims to make built-in browser consistent across all browsers

!important tag specifies attributes that are crucial and must be applied regardless of other specificy

Never use ids! By using ids you lose the cascading feature of CSS and your code becomes very specific and not flexible to changes. I use to always use ids to quickly add CSS.

To center an element. Add width and margin: 0 auto.
