Thursday, April 2, 2015

Why jQuery over straight-up JS? Makes hard things easier to do, code is more concise, less code to write, cross-browser compatibility, and utility.

Everything you deal with in JS is an object: an array is an object, an Oblect is an object, a function is an object, etc. - Johnny Benson.

I'm really feeling the whole $('.className') for searches, instead of having to write the whole document.getElementsByClassName("className"). Less to write, and less to memorize. Also, one of my favorite things about using the jQuery search is that the same $('') syntax is used for all searches, with the only change being inside the single quotes.
