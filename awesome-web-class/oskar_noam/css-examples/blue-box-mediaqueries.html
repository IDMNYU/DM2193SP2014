Media Queries

First steps towards responsive web

  • Absolute positioning breaks free elements from the layout flow completely.
  • No longer nudging elements, but placing them specifically using coordinations.
  • Origin is always 0,0 from the top left corner of the "derived" parent.
  • CSS traverses the DOM back up the tree stopping when it finds a parent with positioning to determine origin or until it gets to the very top.
