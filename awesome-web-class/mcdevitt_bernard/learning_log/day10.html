reading response

classical inheritance in javascript was a dense, but interesting read, and it helped to tie my knowledge of C++ better into my limited knowledge of javascript through the common practice of inheritance. javascript's system of multiple inheritance will certainly take some getting used to, but object augmentation sounds like an extremely useful tool to take advantage of in JS's arsenal.

notes

  • reset stylesheets are useful for providing a blank slate
  • different display techniques allow you to shape boxes on pages differently
  • block will take up whole width available, inline only as much width as necessary
  • float objects to put them next to each other, use clear to fix wrapping issues
  • position absolute will keep an object in place regardless of other objects
  • position relative will move an object relative to others
  • position fixed will keep it static on the screen, even when scrolling
