notes

  • twitter bootstrap essentially comes with all the tools to make a site
  • limited knowledge of css and js required to use it, most layouts are prefabs
  • bootstrap sites have a distinctive look because they use the same fonts & icons
  • bootstrap has a vast library of icons, but changing away from defaults is a pain
  • this is because everything is set up to work the prefab way, very difficult to change things in bootstrap
  • only use it when you need an easy way to make a site fast, it limits creative potential on making sites
